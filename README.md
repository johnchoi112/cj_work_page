# 자재 KPI 대시보드

거래처별 미입고율 KPI를 중심으로 `발주총괄진행현황`, `입고현황` 엑셀 파일을 업로드하여 자동으로 그래프를 생성하는 웹 대시보드입니다.

## 핵심 기능
- 거래처별 미입고율(%) 자동 계산
- 발주총괄진행현황 도넛 차트(완료/진행중)
- 입고일 기준 입고 추이 라인 차트
- 거래처 요약 테이블 자동 생성
- 샘플 데이터 버튼으로 즉시 데모 확인

## 사용 방법
1. `index.html` 접속
2. 발주총괄진행현황 업로드 (거래처, 발주수량, 미입고수량, 발주일)
3. 입고현황 업로드 (거래처, 입고수량, 입고일)
4. `업로드 데이터 분석` 버튼 클릭

python3 -m http.server 4173
